# HTML Style Remover

A lightweight, browser-based tool for stripping CSS and style code from HTML.

**[Live Demo](https://jbmassae.github.io/HTMLStyleRemover/html_style_remover.html)**

## Features

- Remove inline `style` attributes
- Remove `<style>` tags and their content
- Remove `class` attributes
- Remove `<link rel="stylesheet">` CSS file references
- Statistics on characters removed and space saved
- Copy cleaned HTML to clipboard or download as a file

## Usage

1. Open the [live demo](https://jbmassae.github.io/HTMLStyleRemover/html_style_remover.html) or open `html_style_remover.html` locally in any browser
2. Paste your HTML into the input box
3. Select which style elements to remove
4. Click **Remove Style Code**
5. Copy or download the cleaned HTML
